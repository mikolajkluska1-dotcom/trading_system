# ARCHITEKTURA SYSTEMU (5 DYWIZJI)

1. **@Frontend** (`agents/Frontend`) - UI/UX React.
2. **@BackendAPI** (`agents/BackendAPI`) - API, Security, Auth.
3. **@AIBrain** (`agents/AIBrain`) - ML, Trading, Whales.
4. **@Database** (`agents/Database`) - SQL, Data management.
5. **@DevOps** (`agents/DevOps`) - Docker, Config (pliki w Root).

Używaj `@NazwaAgenta` w czacie, aby wezwać odpowiedniego specjalistę.
